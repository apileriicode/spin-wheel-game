# 🎡 Spin Wheel Game Bot

Telegram бот для гри у колесо удачі з системою рефералів, платежами в TON/зірках/USDT.

## 🎯 Можливості

- ✅ **Прокрут колеса** - ставка від 1 до 999,999,999 зірок
- ✅ **Система рефералів**:
  - +3 зірки за запрошення друга
  - +2 зірки якщо друг поповнить рахунок
- ✅ **Валюти**: TON, зірки, USDT
- ✅ **Профіль користувача** з статистикою
- ✅ **Лідерборд** топ гравців
- ✅ **База даних** SQLite

## 🚀 Встановлення

```bash
# Клонувати репозиторій
git clone https://github.com/apileriicode/spin-wheel-game.git
cd spin-wheel-game

# Встановити залежності
pip install -r requirements.txt

# Налаштувати .env
cp .env.example .env
# Додайте TELEGRAM_BOT_TOKEN від @BotFather

# Запустити бот
python main.py
```

## 📋 Команди

- `/start` - Почати гру
- `/spin` - Крутити колесо (вказати суму)
- `/profile` - Мій профіль
- `/ref` - Мої реферали
- `/leaderboard` - Лідерборд
- `/help` - Допомога

## 💰 Ставки на колесо

- **Мінімум**: 1 зірка
- **Максимум**: 999,999,999 зірок
- **Приз**: випадковий від 0.5x до 5x від ставки

## 🔧 Налаштування

Відредагуйте `config.py`:
```python
SPIN_COST = 100  # Вартість одного прокруту
REFERRAL_BONUS = 3  # Бонус за запрошення
REFERRAL_DEPOSIT_BONUS = 2  # Бонус за поповнення реферала
```
