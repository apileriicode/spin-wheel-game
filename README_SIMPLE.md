# 🎡 Simple Earning Bot

Простий Telegram бот для гри та заробітку!

## 🚀 Швидкий Початок

### 1. Данованя у Replit

```bash
# 1. Перейти на https://replit.com
# 2. Натисніть "Створити Repl"
# 3. Вибрати "Import from GitHub"
# 4. Поставити apileriicode/spin-wheel-game
```

### 2. Налаштування

```bash
# Налаштуюеме оточення
# Перейти: Secrets (🔐 іконка)
# додати:
TELEGRAM_BOT_TOKEN=your_token_here
```

### 3. Проведення

```bash
pip install -r requirements_simple.txt
python simple_bot.py
```

## 🎡 Гри у боті

1. **🎡 Кликните кнопку** - 10-100 ⭐
2. **🎢 Вагайте** - 20-150 ⭐
3. **🎰 Крутнуті колесо** - 50-500 ⭐
